# Welcome to the 2026 Workday DevCon Hackathon!

## Product Statement :warning:

Any unreleased services, features, functionality or enhancements referenced in any Workday document, roadmap, blog, our website, press release or public statement that are not currently available are subject to change at Workday’s discretion and may not be delivered as planned or at all.

Customers who purchase Workday services should make their purchase decisions based upon services, features, and functions that are currently available.

## Get Started :muscle:

Join the [Workday DevCon Hackathon](https://devconhackathon.slack.com) Slack workspace to stay informed and get help from an expert by [submitting a ticket](https://devconhackathon.slack.com/archives/C08R43B6TMM).

Judges criteria include: **Use of Platform**, **Innovation**, **Business Value**.

Submissions this year include: **5-minute verbal presentation to first-round judges. (See details below).** * *We encourage you to take advantage of the Pitch Coaches available at the Experts table to refine your presentation.* *

**Optional**: 5-minute recorded video (submitted in Google Form) within two weeks *after* the event.

## Recently Released to Production :tada:

| Feature | Description | 
| ------- | ----------- | 
| [Workday Developer CLI](docs/recently-released-features/wdcli.md) | WDCLI is your unified command line interface to build, test, validate, and deploy Workday Integration and Extend apps. | 
| [Local Disk Sync](docs/recently-released-features/local-disk-sync.md) | Enjoy seamless, automatic syncing between the builder tools in the browser and your local file system- enabling smoother integration with your preferred version control systems and CI/CD workflows while accelerating your development cycles. | 
| [Orchestrate - Kill Orchestration](docs/recently-released-features/orchestrate-kill-orchestration.md) | This new emergency operation gives developers the ability to forcefully and immediately terminate a running orchestration, thus bypassing all cancellation checkpoints when a critical incident demands it. | 
| [Orchestrate - Wait for Long Running API Responses](docs/recently-released-features/orchestrate-wait-for-long-running-api-responses.md) | This feature allows orchestrations to wait for Workday REST and SOAP API requests to complete beyond the standard 5-minute response timeout, supporting up to 1 hour for REST API requests and up to 6 hours for SOAP API requests. | 
| [Extend AI Widgets](docs/recently-released-features/extend-ai-widgets.md) | This feature allows you to embed custom generative AI prompts directly into your application’s rich text interfaces. |

## Additional App Catalog Samples :bulb:

In addition to the samples listed above, we want to remind everyone that the [App Catalog](https://developer.workday.com/app-catalog) on the Developer Site remains a helpful resource for many sample patterns and foundational building blocks for your Extend applications and Orchestrations. Here are a few noteworthy samples that may prove to be helpful during the Hackathon:

| App | Description | 
| --- | ----------- | 
| [PMD Widget Dictionary](https://developer.workday.com/app-catalog/pmdWidgetDictionary) | Get to know a wide variety of Presentation Components tags, all demonstrated in this app. | 
| [PMD Scripting](https://developer.workday.com/app-catalog/pmdScripting) | See some of the most common uses of PMD Scripting in action. | 
| [Workday Orchestrate Sampler](https://developer.workday.com/app-catalog/orchestrateForIntegrationsSampler) | Find quick examples that demonstrate Orchestrate functionality. | 
| [Extend Orchestration Sampler](https://developer.workday.com/app-catalog/orchestrationToolkit) | Take advantage of reusable patterns and reference orchestrations for many different Extend use cases. | 
| [Orchestrate Insight APIs](https://developer.workday.com/app-catalog/orchestrateInsights) | This app highlights how to use the Orchestate Insights APIs by using an Extend App to execute and display the results. The app includes an Integration Orchestration that can extract large volumes of Orchestrate events which can be used to export to an analytics tool. | 
| [AWS Integration Starter Kit](https://developer.workday.com/app-catalog/awsStarterKit) | Quickly integrate AWS AI APIs using AWS Lambdas with this sample starter kit. | 
| [AWS Badge Generator](https://developer.workday.com/app-catalog/AWSBadgeGenerator) | Allow employees to get a photo badge with AI-powered face detection and image generation with Amazon Web Services (AWS). | 
| [Prism and Extend Design Patterns](https://developer.workday.com/app-catalog/prismAndExtendDesignPatterns) | This app highlights the importance for designing scalable pages when interacting with large data sets. It also highlights a single threaded pattern for triggering Prism Data Change Tasks (DCT) | 
| [Sales Commission - A Workday Build Prototype](https://developer.workday.com/app-catalog/salesCommission) | This app ingests external sales data into Workday Prism to automate commission calculations. It then surfaces those results in Workday Extend for organizational review and manual adjustments. | 
| [Crisis Outreach](https://developer.workday.com/app-catalog/crisisCheckin) | A simple, well-written Extend app with examples of model components. | 
| [Create A Work Event With AI Widget](https://developer.workday.com/app-catalog/createAWorkEventWithAIWidget) | Enable employees to propose and create work events, to register for events, and use a calendar view. | 
| [JavaScript Example](https://github.com/Workday/extend-js-example) | Build a custom JavaScript UI that interacts with Workday Extend apps. Pairs well with the Spot Bonus and Worker Badges app. |
