The SFTP File Transport component enables you to perform a number of SFTP operations, dynamically determining paths at runtime.

Use the Create Integration CredStore task in your tenant to configure SFTP credentials. You can then reference that CredStore in an orchestration’s Settings panel and subsequently use the credentials with the new component.
