**What’s New?**

1. You can now access and analyze performance summary metrics for each of your orchestration runs straight from the Orchestration Builder experience.
2. You can also access this analysis from each of your runs in the Orchestrate Activity Dashboard. Clicking that link will bring you straight to the performance summary in Orchestrate Builder.

**Capabilities and Benefits**

1. Any user can quickly and simply understand the performance summary of their Orchestration runs directly in Orchestration Builder.
2. Allows developers to understand which Orchestration is running longest, why it is running long, and what specific part is running long.
3. Analyze and sort by successes, failures, avg deviation, min/max duration, total, and std deviation.
4. Ultimately, once the target improvement is found, you are able to resolve that issue directly in that builder tool, right alongside the metrics. No need to leave the builder tool to analyze and resolve, all in one place.
5. Also accessible through Activity Dashboard.

**Availability**

As long as you have access to build Orchestrations, you can use the Summary Metrics feature in Orchestration Builder. At Devcon, the feature will ONLY be available at the Hackathon. More to come on availability in Prod post Devcon.
