This documentation was prepared for Workday DevCon 2026. It assumes your familiarity with Workday Studio and is not intended as a complete guide to Studio integration development. Instead, it provides an overview of the Studio Builder tool as a distinct entity, enabling you to quickly employ your experience as a Studio developer in a new environment.

Full documentation for Workday Studio is available in the Workday Adminstrator guide [here](https://doc.workday.com/admin-guide/en-us/workday-studio/setup-considerations--workday-studio.html?toc=0).

> **Note:** The version of Studio Builder available at Workday DevCon 2026 is a work in progress and not fully representative of the features expected in the finished product.
