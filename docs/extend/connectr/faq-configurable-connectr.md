### Can I have my Extend task conditionally appear only for certain users or scenarios?

Conditional logic must be implemented within the Extend task itself (e.g., show/hide based on domain security). The ConnectR configuration does not support conditional section rendering at this time

### What happens if a user navigates away from my Extend section without saving?

Data entered in an Extend section is submitted to the business object upon navigation.

### Can I use the same Extend task in multiple flows?

Yes, the same Extend edit task can be added as a section to different Configurable ConnectR configurations independently.

### Are there limits on how many Extend tasks can be in one flow?

Yes a maximum of 5 custom sections (including Extend sections) can be added to a single flow definition via Configure ConnectR.
