# Analytics Agent

## Overview

The Analytics agent is a conversational AI agent that enables managers, business  
leaders, and HR and Financial users to perform reporting actions in Workday. You can  
use the agent to:  
● Analyze and explore data to answer specific questions about enabled reports.  
● Identify and analyze trends to spot data patterns over time and highlight  
significant shifts, provided the report data supports it.  
● Summarize report outputs to generate concise easy-to-read overviews of  
complex datasets.

## Business Benefits

The Analytics agent simplifies data access and improves data-driven decision-making  
by providing automated data exploration, visual insights, and information  
summarization.  

The agent saves time for a wide range of users by providing immediate answers to  
complex data inquiries without requiring technical report-building skills. It uses your  
current context within the application and enforces existing security permissions,  
ensuring that you only access authorized data based on your assigned security groups.

## Limitations

Consider these factors when using the Analytics agent:  
● Supported Report Types: You can only enable the agent to analyze data for  
Advanced reports.  
● Data Limits: The agent analyzes report output sizes up to a specific limit. See:  
Reference: Reporting Limits.  
● Report Execution: The agent only supports analyzing reports that are in the  
Workday interface.

# Data Analysis and Exploration Skill

## Overview

The Data Analysis and Exploration skill of the Analytics agent provides contextual
data exploration and insights for your Workday reports. This skill enables you to get
answers to your data questions without needing to understand complex reporting tools
or underlying data structures.

When you query the agent, Workday uses conversational AI to generate automated
summaries and explanations that identify trends and patterns. The skill supports
multi-turn dialogues, allowing you to ask follow-up questions to further explore the report
data. This skill is only applicable for Advanced reports.

## How to Access

Before you can access this skill, you must configure the necessary data and
permissions.

Access Agent Chat while viewing an enabled custom report and enter a query related
to the report data.

## Limitations

Consider these factors when using the Analytics agent:
● Query Scope: The agent provides analysis based strictly on the specific report
data it is enabled on. It does not answer subjective questions.
● Data Analysis: You can only use this skill to analyze data within Advanced
reports.

## Example Queries

● Summarize this report.
● What were the top 3 expense categories for the Marketing department last
month?
● Summarize my team’s performance for last week.
